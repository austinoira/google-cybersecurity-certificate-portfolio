## Certificate Overview

The program covers eight courses spanning foundational security concepts through hands-on technical skills including Linux, SQL, Python scripting, SIEM tools, and network analysis.

**Key skill areas:**
- Security frameworks and risk management (NIST CSF, CIA Triad)
- Network architecture and traffic analysis (Wireshark, tcpdump)
- Linux command line and file permissions
- SQL for security investigation
- Incident detection and response (SIEM, IDS)
- Python automation for security tasks
- Asset classification and vulnerability management
